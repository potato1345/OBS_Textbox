# OBS_Textbox

OBS browser view text field that can be customized via the "Custom CSS" tab.

## Preview

![OBS Textbox Preview](preview.png)

## Setup 

to configure the Textbox, paste the following Code in your OBS Custom-CSS Settings, and paste the text in between the corresponding quotation marks like this:

```css
:root {
    --zeile-1: "erste Zeile";
    --zeile-2: "zweite Zeile";
    --zeile-3: "dritte Zeile"; 
}
```
